# 🚀 بوت Velora Gate الرقمي

بوت تيليجرام متطور لبيع الحسابات الرقمية (Apple ID, Gmail) مع لوحة تحكم شاملة للمسؤولين

## ✨ المميزات

### 🎯 للمستخدمين العاديين:
- 📦 عرض المنتجات المتاحة
- 💰 عرض الأسعار المحدثة
- 💳 معلومات طرق الدفع
- 💱 سعر صرف العملات
- 🤝 خدمة الدعم الفني

### 👨‍💼 لوحة التحكم (للآدمن):
- 📊 إحصائيات المستخدمين والإيرادات
- 💰 تعديل أسعار المنتجات مباشرة
- 💳 تحديث طرق الدفع
- 📢 إرسال رسائل جماعية
- ✏️ تعديل جميع الرسائل
- 🗄️ قاعدة بيانات SQLite

## 📋 المتطلبات

```bash
pip install -r requirements.txt
```

## ⚙️ التثبيت

### 1. استنساخ المستودع
```bash
git clone https://github.com/englishr222-wq/Velora-Gate-.git
cd Velora-Gate-
```

### 2. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 3. إعداد config.py
افتح `config.py` وأضف:
- رمز البوت (BOT_TOKEN)
- معرفات الآدمن (ADMIN_IDS)

```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"
ADMIN_IDS = [123456789]  # أضف معرفك أنت
```

### 4. تشغيل البوت
```bash
python main.py
```

## 🔐 كيفية الوصول إلى لوحة التحكم

### للآدمن:

1. **من القائمة الرئيسية:**
   - اضغط على زر "👨‍💼 لوحة التحكم"

2. **باستخدام الأمر:**
   ```
   /admin
   ```

### المتطلب الأساسي:
- يجب أن يكون معرف الآدمن موجود في `ADMIN_IDS` في `config.py`

## 📊 خيارات لوحة التحكم

| الخيار | الوصف |
|--------|-------|
| 📊 الإحصائيات | عرض عدد المستخدمين والإيرادات الكلية |
| 💰 تعديل الأسعار | تغيير أسعار Apple ID و Gmail |
| 💳 تعديل طرق الدفع | تحديث تفاصيل الحسابات البنكية |
| 📢 رسالة جماعية | إرسال إعلانات لجميع المستخدمين |
| ✏️ تعديل الرسائل | تخصيص رسائل الترحيب والدعم |

## 🗄️ قاعدة البيانات

يتم حفظ البيانات تلقائياً في `velora_gate.db` تتضمن:

- **المستخدمون**: المعرف، الاسم، تاريخ الانضمام، عدد المشتريات
- **الأسعار**: أسعار المنتجات المحدثة
- **طرق الدفع**: معلومات الحسابات البنكية
- **الرسائل**: الرسائل المخصصة
- **المعاملات**: سجل العمليات
